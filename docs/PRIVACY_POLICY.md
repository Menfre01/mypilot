# 隐私政策 / Privacy Policy

**最后更新 / Last Updated**: 2026-04-16

---

## 中文 / Chinese

### 数据收集

MyPilot 不收集、存储或向外部服务器传输任何个人用户数据。

### 权限使用

| 权限 | 用途 |
|---|---|
| 相机 | 仅用于扫描二维码以配对 Gateway。不拍摄或存储照片或视频。 |
| 本地网络 | 仅与用户自己的 Gateway 服务器进行本地网络通信。无需互联网访问。 |

### 数据存储

所有用户数据均使用系统提供的键值存储（SharedPreferences）存储在设备本地。数据不会上传或与第三方共享。

### 第三方服务

MyPilot 不集成任何第三方分析、广告或跟踪服务。

### 儿童隐私

MyPilot 不会故意收集儿童的个人信息。本应用适合所有年龄段（4+）。

### 变更

我们可能会不时更新本政策。任何变更都将在此页面反映并更新日期。

### 联系方式

如需隐私相关咨询，请联系：menfre@proton.me

## English

### Data Collection

MyPilot does not collect, store, or transmit any personal user data to external servers.

### Permissions

| Permission | Purpose |
|---|---|
| Camera | Scan QR codes for Gateway pairing only. No photos or videos are captured or stored. |
| Local Network | Communicate with the user's own Gateway server on the local network. No internet access required. |

### Data Storage

All user data is stored locally on the device using system-provided key-value storage (SharedPreferences). No data is uploaded or shared with third parties.

### Third-Party Services

MyPilot does not integrate any third-party analytics, advertising, or tracking services.

### Children's Privacy

MyPilot does not knowingly collect information from children. The app is suitable for all ages (4+).

### Changes

We may update this policy from time to time. Any changes will be reflected on this page with an updated date.

### Contact

For privacy-related inquiries, contact: menfre@proton.me
